# scrabble-timer
This project features a online HTML Scrabble timer, which may be useful for your Scrabble games!

A digital Scrabble timer with features like:

- SPACEBAR to switch players  
- Red warning at 30 seconds  
- Beep sound when time ends  
- Move counter for turns  
- Fullscreen mode  
- Seven-segment LED display  
- Large paddle buttons  

All files needed for the timer are included in this repository.  

Created by **Gaurangaa** for fun family game nights and Scrabble games!
